# C-Chat Server Backend

## 功能

- 用户认证与注册系统
- 数据库连接
- 基于Boost.Asio的异步网络通信
- 多线程支持
- 用户会话管理

## 技术栈

- C++
- Boost.Asio网络库
- MySQL数据库
- 多线程编程

## 系统要求

- C++11或更高版本
- Boost库 (1.66.0或更高版本)
- MySQL服务器
- CMake 3.10+
